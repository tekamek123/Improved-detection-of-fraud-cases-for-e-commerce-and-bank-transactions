# Reports Directory

This directory contains generated reports and analysis outputs.

## Contents

### Interim Reports
- `interim_report_task1_*.pdf`: Task 1 interim reports with data analysis, EDA insights, feature engineering, and class imbalance analysis

### Future Reports
- Model comparison reports
- Final project report
- SHAP explainability reports

## Report Generation

Reports are generated using Python scripts in the `scripts/` directory:

```bash
# Generate Task 1 interim report
python scripts/generate_interim_report.py
```

## Report Structure

Reports typically include:
- Executive summary
- Data cleaning and preprocessing steps
- Exploratory data analysis insights
- Feature engineering rationale
- Model performance metrics
- Visualizations and tables
- Next steps and recommendations

## Notes

- Reports are generated as PDF files for easy sharing
- All visualizations and tables are included in the PDF
- Reports are timestamped for version tracking
- Old reports are kept for reference and comparison
